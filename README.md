# rboxer

### [leaflet-routerboxer](https://github.com/StephanGeorg/leaflet-routeboxer)  for node.js

#### install

```
  npm install rboxer --save
```

#### usage

```
  const rboxer = require('rboxer')

  const pts = [[10,10],[11,12],[12,11],[13,14],[14,13]]
  const boxes = rboxer.mkBox(pts, 5)
```
